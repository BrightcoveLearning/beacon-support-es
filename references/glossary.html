Descripción general

Se necesita una cuenta de Stripe para permitir los pagos de los clientes para las siguientes plataformas:

  • Web
  • Samsung Smart TV
  • Sony Smart TV
  • Toshiba Smart TV

Esto significa que Stripe es necesario para la aplicación web y todas las aplicaciones de Smart TV. otro que Android TV y Fire TV.

Los siguientes pasos proporcionan una descripción general del proceso de uso de Stripe con Brightcove Beacon:

  1. Cree su cuenta proporcionando información básica (correo electrónico, nombre, contraseña).
  2. Active su cuenta proporcionando información sobre su negocio.
  3. Recupera tus claves de API de Stripe.
  4. Proporcione al personal de Brightcove las claves de API para que puedan agregarse a Brightcove Beacon Experience.

Crear y activar la cuenta de Stripe

  1. Busque el Página de inicio de Stripe.
  2. Haga clic en el EMPEZAR AHORA botón.
  3. Complete su información básica para crear la cuenta.
  4. Asegúrese de que se le haya mostrado el panel de Stripe.
  5. Haga clic en el Activa tu cuenta de Stripe opción.
  6. Haga clic en el Empezar ahora para comenzar el proceso.
  7. Deberá verificar su dirección de correo electrónico.
  8. Después de verificar su dirección de correo electrónico, verá el formulario que debe completar para activar su cuenta. Destacadas en amarillo están las secciones principales del formulario.
  9. Después de completar el formulario, haga clic en el Presentar la solicitud botón.

Obtener sus claves de API de Stripe

La conexión entre su cuenta de Stripe y Brightcove Beacon se realiza a través de las claves API de Stripe. En la captura de pantalla del Panel de control que se muestra arriba, verá un enlace a Obtenga sus claves API en vivo. Tenga en cuenta que hasta que se active una cuenta, solo podrá obtener claves API de prueba.

En el panel, haga clic en el Obtenga sus claves API en vivo y siga las instrucciones para descargarlos.

Conexión de Stripe a Brightcove Beacon

Una vez que tenga sus claves de API de Stripe, se las entregará al personal de Brightcove que está configurando su Brightcove Beacon Experience. Asociarán sus claves de API de Stripe con su Brightcove Beacon Experience para que cuando configure Stripe para que se use en sus aplicaciones, se asocie con su cuenta.
