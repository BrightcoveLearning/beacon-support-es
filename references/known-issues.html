Problemas conocidos

Este tema enumera los problemas conocidos de {{ site.product }} solicitud.

Aplicaciones generadas

Limitaciones de transmisión y cambio rápido entre videos

  1. Si un espectador cambia rápidamente entre videos, es posible que se muestre el error que se muestra aquí:

    Esto ocurre porque la transmisión tarda varios segundos en liberarse y, si el espectador intenta ver otro video antes de que se publique, se produce una limitación de la transmisión y se muestra el error.

Controles parentales

  1. Todo el contenido se muestra a los usuarios finales (en listas, carruseles, etc.), pero al intentar reproducir el video se solicitará el pin parental.

Web móvil

  1. La web móvil tiene soporte limitado. En dispositivos con un ancho inferior a 375 px, se muestra un mensaje para instalar la aplicación nativa Brightcove Beacon adecuada.
