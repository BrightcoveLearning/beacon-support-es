Descripción general

Para crear una configuración de publicidad, deberá hacer lo siguiente:

  • Obtenga una URL del servidor de etiquetas publicitarias de su proveedor de servicios publicitarios. Esto se utilizará en Video Cloud Studio para crear su configuración de anuncios SSAI.
  • En Video Cloud Studio, cree su configuración de anuncios SSAI. Al finalizar la tarea, copiará el ID del servidor de anuncios para crear su configuración de anuncios en Brightcove Beacon Classic.
  • En Brightcove Beacon Classic cree su configuración de publicidad, que luego podrá aplicar a su contenido de video.

Cree un ID de configuración de anuncios SSAI

Siga estos pasos para crear una configuración de anuncios SSAI:

  1. En Video Cloud Studio, navegue hasta Admin → Configuración de anuncios del lado del servidor.
  2. Desde el Crear configuración de anuncios desplegable, seleccione VOD
  3. Agregue un nombre para la configuración de anuncios.
  4. Seleccione la respuesta de su anuncio: se recomienda VMAP o VAST. Si usa VAST, deberá agregar puntos de referencia a los videos para especificar dónde deben aparecer las pausas publicitarias.
  5. Pegue su etiqueta de anuncio.
  6. Haz clic en Guardar.
  7. Busque su nueva configuración de anuncios en la tabla y copie su ID de configuración de anuncios. Necesitará este valor para la configuración de publicidad de Beacon.

Crear una configuración de publicidad

Para crear una configuración de publicidad, siga estos pasos:

  1. En Brightcove Beacon Classic, en la barra de navegación superior, seleccione Anuncio publicitario .
  2. Hacer clic Agregar nueva configuración.
  3. Nombre la configuración de anuncios.
  4. Para el Proveedor Seleccione Brightcove SSAI
  5. Pegar en el ID de configuración de anuncio SSAI copiado en los pasos anteriores en ID de configuración de anuncios de VOD.
  6. Hacer clic Crear nuevo anuncio.
