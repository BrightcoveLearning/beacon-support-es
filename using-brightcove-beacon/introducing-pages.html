Categorías de página

Los tipos de página se pueden agrupar en tres categorías. Las categorías y los detalles de cada uno son los siguientes:

  1. No configurable
    • Una página que eliges incluir en tu aplicación, o no.
    • El documento Incluyendo páginas no configurables proporciona más detalles, incluido cómo agregar uno de estos tipos de páginas a su aplicación.
    • Ejemplos de páginas no configurables son Ajustes y Inscribirse , mostrado aquí:
  2. Pantalla
    • Una página que contiene su contenido de video, en carruseles, listas, etc.
    • Altamente configurable usando bloques , diseños y listas de reproducción. Se muestran dos ejemplos en plataformas móviles, seguidos de un ejemplo de aplicación web:
    • screen-hero_image
      • Un tipo especial de página de pantalla, pero permite una imagen detrás del contenido del video.
      • Necesita cargar una imagen de orientación horizontal y de póster para diferentes plataformas, por ejemplo, web (horizontal) y móvil (vertical).
      • La "héroe" El área en la parte superior de la pantalla está destinada a no estar cubierta y no se puede configurar:
  3. Redirigir
    • Los tipos de página de redireccionamiento son solo para la aplicación web.
    • redireccionar-en blanco abre una página web en una nueva pestaña.
    • redirigir a uno mismo abre una página web en la misma pestaña.
    • Especifique la URL en el Pestaña de redireccionamiento externo.

Configurar páginas de pantalla

Como se mencionó anteriormente en este documento, pantalla Las páginas contienen sus videos y brindan opciones para la visualización de las imágenes que representan sus videos. En esta sección, verá sus opciones de configuración con capturas de pantalla de cómo se ven esas opciones en sus aplicaciones. Primero verá información sobre bloques, que es el nivel más alto de contenedor en una página de pantalla. Luego verá información de diseño, que controla cómo se muestran las imágenes que representan sus videos.

Bloques

Destacados en amarillo están los tipos de bloques que puede utilizar:

Aquí hay imágenes de cada tipo:

bloque único

El bloque único solo le permite agregar una combinación de diseño / lista de reproducción al bloque. La captura de pantalla muestra un bloque único mostrando un carrusel. Tenga en cuenta que si intenta agregar varias combinaciones de diseño / lista de reproducción a un bloque únicoo intente cambiar a un bloque único que contiene múltiples combinaciones de diseño / lista de reproducción, obtendrá un error:

bloque de lista

El bloque de lista le permite agregar múltiples combinaciones de diseño / lista de reproducción al bloque. La captura de pantalla muestra un bloque de lista con tres carruseles contenidos en el mismo.

tabed_block

El bloque-tabulado le permite agregar múltiples combinaciones de diseño / lista de reproducción al bloque, pero solo se mostrará una a la vez y las pestañas le permiten al usuario elegir cuál es visible. Se aplican dos restricciones:

  • Los diseños dentro de un bloque con pestañas deben ser los mismos.
  • Dentro de un bloque con pestañas, los diseños deben ser cuadrículas.
La captura de pantalla muestra un bloque-tabulado con listas de reproducción seleccionables.

bloque-desplegable

El bloque-desplegable le permite agregar múltiples combinaciones de diseño / lista de reproducción al bloque, pero solo se mostrará una a la vez y el menú desplegable le permite al usuario elegir cuál es visible. Se aplican dos restricciones:

  • Los diseños dentro de un bloque desplegable deben ser los mismos.
  • Dentro de un bloque desplegable, los diseños deben ser cuadrículas.
La captura de pantalla muestra un bloque-desplegable (dentro del rectángulo amarillo) con tres listas de reproducción seleccionables.

Diseños

En todos los ejemplos de bloques sobre el diseño seleccionado fue un carrusel. El carrusel muestra la imagen de cada video en una lista horizontal. Ese no es el único diseño entre el que puede elegir. Los diseños disponibles están resaltados en amarillo:

En lugar de una captura de pantalla de cada tipo de diseño, se mostrará un ejemplo de cada característica de diseño. Los tipos resaltados en amarillo arriba son combinaciones de diferentes características.

Carrusel y etiquetas en

El carrusel muestra imágenes en una fila horizontal y las etiquetas se superponen a una pequeña parte inferior de la imagen.

Etiquetas hacia fuera

Las etiquetas están debajo de la imagen.

Presentado

Las imágenes son más grandes (aparecen en un cuadro rojo, carrusel normal debajo de eso). A menudo, pero no siempre, los diseños destacados se utilizan en la parte superior de una página.

Paisaje

Las imágenes son de proporciones paisajísticas.

Carteles

Las imágenes son póster (retrato) proporcionado.

Cuadrícula 3 Y Cuadrícula 4

Los dos tipos de cuadrículas muestran imágenes en un patrón de fila / columna bidimensional. El número que sigue a la cuadrícula, 3 o 4, indica un tamaño relativo para las imágenes. NO indica el número de columnas que siempre se mostrarán. Esta captura de pantalla muestra un navegador de 800 píxeles de ancho y Grid 3 tiene dos filas de tres imágenes. El Grid 4 muestra cuatro imágenes a lo ancho ya que las imágenes son más pequeñas.

La siguiente captura de pantalla todavía usa Grid 3 y Grid 4, pero el ancho del navegador es de 1600px. Las imágenes de Grid 3 ahora tienen cuatro de ancho, y cinco imágenes en el diseño de Grid 4 pueden caber en la pantalla. De nuevo, puede ver que las imágenes de la cuadrícula 3 son más grandes.

Eso concluye las opciones para bloques y diseños.
