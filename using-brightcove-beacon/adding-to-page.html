Agregar un carrusel a una página existente

En este tema, aprenderá a agregar un grupo de videos en un carrusel a una página de Brightcove Beacon.

Descripción general

La página de Brightcove Beacon es donde los usuarios seleccionan y miran su contenido de video. Los carruseles muestran un grupo de videos o listas de reproducción que ha definido en Video Cloud Studio. Con la Consola administrativa de Brightcove Beacon, puede agregar carruseles a una página existente.

Agregar un carrusel

Para agregar un carrusel a una página existente, siga estos pasos:

  1. Inicie sesión en Brightcove Beacon.

  2. En la navegación superior, seleccione Paginas.

  3. Puedes usar el Agregar nueva página botón para crear una nueva página. Una vez que haya definido las páginas, puede agregar fácilmente un carrusel a una página existente. Seleccione la página que desea actualizar. Aquí, estamos seleccionando el enlace para el Casa página.

  4. Note que el Datos básicos ya está poblado.

  5. Selecciona el Agregar bloque botón.

  6. Elija el tipo de bloque que desee. En este caso, estamos eligiendo el bloque de lista. Selecciona el Agregar bloque botón asociado al bloque que desee.

  7. Selecciona el Agregar diseño botón.

  8. Elija el tipo de diseño que desee, ya sea Destacado, Carrusel o Cuadrícula. En este ejemplo, estamos seleccionando el Etiquetas de paisaje de carrusel en diseño.

  9. Ahora que tiene el contenedor definido para el carrusel, está listo para agregar contenido. En el elemento de bloque de lista, seleccione el Agregar lista de reproducción botón.

  10. Elige una lista de reproducción. En este ejemplo, estamos seleccionando el Agregar lista de reproducción botón asociado con el Humor lista de reproducción.

    Las listas de reproducción se crean con Video Cloud Studio. Aparecerán aquí una vez que se sincronice con su cuenta de Video Cloud.

  11. Podemos ver que el Humor La lista de reproducción se ha agregado a nuestro carrusel. Puede arrastrar y soltar para reordenar los elementos. Selecciona el Actualizar página para agregar el nuevo carrusel a la página.

  12. Estas listo. Debería ver el nuevo carrusel en su página de Brightcove Beacon. Consulte las siguientes secciones para obtener detalles sobre los valores de campo.
