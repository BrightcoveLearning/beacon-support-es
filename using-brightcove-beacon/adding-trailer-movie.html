Agregar un tráiler

Agregar un avance para una película de Beacon es simple. Siga los pasos a continuación.

  1. Cargue el video de avance a su cuenta de Video Cloud.
  2. Tú haces no Necesita agregar campos personalizados especiales de Beacon al video de avance. Todo lo que necesitas es el ID de video:
  3. Para el video que se usa para la película Beacon, agregue el siguiente campo personalizado:
    Campo personalizado de tráiler
    Nombre para mostrar Nombre interno Valor
    beacon.trailer.id beacon_trailer_id (ID de video para el avance)
