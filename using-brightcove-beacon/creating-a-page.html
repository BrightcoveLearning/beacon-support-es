Crear una nueva página

En este tema, aprenderá a crear una nueva página de Brightcove Beacon.

Descripción general

La página de Brightcove Beacon es donde los usuarios seleccionan y miran su contenido de video. Los carruseles muestran un grupo de videos o listas de reproducción que ha definido en Video Cloud Studio. Con la Consola administrativa de Brightcove Beacon, puede crear una nueva página con un carrusel de contenido.

Creando una pagina

Para crear una nueva página, siga estos pasos:

  1. Inicie sesión en Brightcove Beacon.

  2. En la navegación superior, seleccione Paginas.

  3. Selecciona el Agregar nueva página botón para crear una nueva página.

  4. Para el Datos básicos pestaña, introduzca los siguientes valores de campo:

    • Título del sistema - Nombre de la página del sistema interno. Esto no se mostrará a los usuarios, pero se usa para generar una URL amigable. Ingresará un título visible en un paso posterior.
    • Pedido - Orden de las páginas en su página de inicio
    • Tipo de género - Valores:
      • Maestría - Pagina principal
      • Niño - Página subordinada
    • Tipo de navegación - Seleccionar de una lista de plantillas de página
    • Tipo de presentación - Valores:
      • icono de texto - Mostrar un icono con texto dentro
      • icono - Mostrar un icono
      • texto - Mostrar texto
    • Diseño de pantalla - Valores: (no todos los tipos de navegación tendrán un diseño de pantalla)
      • Diseño de pantalla titulado - Diseño con títulos
      • Diseño de pantalla sin título - Diseño sin títulos
    • Esta en casa? - Seleccione si esta es la página de inicio
  5. A continuación, agregaremos un bloque a la página. Los bloques son contenedores para tus listas de reproducción. Algunos tipos de navegación no tienen bloques, como los tipos de inicio de sesión y cierre de sesión. Hemos seleccionado un Tipo de navegación con un valor de pantalla.

  6. Selecciona el Agregar bloque botón. Cuando seleccione el botón, los tipos de bloque aparecerán en el lado derecho.

  7. Elija el tipo de bloque que desee. En este caso, estamos eligiendo el bloque de lista. Selecciona el Agregar bloque botón asociado al bloque que desee.

  8. Selecciona el Agregar diseño botón. Cuando seleccione el botón, los tipos de diseño aparecerán en el lado derecho.

  9. Elija el tipo de diseño que desee, ya sea Destacado, Carrusel o Cuadrícula. En este ejemplo, estamos seleccionando el Etiquetas de paisaje de carrusel en diseño.

  10. Selecciona el Agregar lista de reproducción botón. Cuando seleccione el botón, los tipos de lista de reproducción aparecerán en el lado derecho.

  11. Elige una lista de reproducción. En este ejemplo, estamos seleccionando el Agregar lista de reproducción botón asociado con el Humor lista de reproducción.

    Puede crear listas de reproducción con Video Cloud Studio y aparecerán aquí una vez que las sincronice con su cuenta de Video Cloud.

  12. Podemos ver que el Humor La lista de reproducción se ha agregado a nuestro carrusel. Puede arrastrar y soltar para reordenar los elementos.

  13. Para el Datos textuales , Datos no textuales , Etiquetas y Derechos y programación pestañas, ver el Parámetros de configuración comunes documento.

  14. Selecciona el Crear nueva página botón para crear su nueva página. Estas listo. Debería ver el nuevo carrusel en su página de Brightcove Beacon.
